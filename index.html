Using the Print Button

Each form includes a print button that allows you to save or print a copy for record-keeping or submission. Follow these steps:

  • After filling out the form, click the Print button located at the bottom of the page.
  • In the print dialog that appears, select Save as PDF from the printer options if you want to save a digital copy instead of printing it.
  • To ensure the form fits neatly on one page, use the Custom Scaling option:
    • Find the scaling or page size settings in the print dialog (usually under "More Settings").
    • Adjust the scaling percentage to shrink or expand the form to fit within the page margins.
    • Preview the form to ensure everything is properly aligned.
  • Once satisfied, click Print or Save to finalize the process.
